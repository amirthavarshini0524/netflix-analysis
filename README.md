# 🎬 Netflix Data Analysis Project

## 📌 Project Overview
This project is an exploratory data analysis (EDA) of Netflix movies and TV shows dataset. The goal is to understand content distribution, trends, and key insights such as genre focus, movie duration patterns, and comparison between Movies and TV Shows.

---

## 📊 Objectives
- Analyze the distribution of Movies vs TV Shows
- Identify top genres on Netflix
- Study movie duration patterns and detect outliers
- Understand Netflix’s content strategy and focus areas
- Extract meaningful business insights from the dataset

---

## 📁 Dataset
The dataset contains information about Netflix titles including:
- Title
- Type (Movie / TV Show)
- Director & Cast
- Country
- Release Year
- Duration
- Genre (Listed In)
- Rating

---

## 🛠 Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Exploding genre column for analysis
- Statistical summary of movie durations
- Outlier detection using IQR method
- Visualization of:
  - Movies vs TV Shows distribution
  - Top genres
  - Longest movies
  - Duration patterns

---

## 🔍 Key Insights
- Movies dominate Netflix’s content library compared to TV Shows.
- Netflix focuses heavily on popular genres like Drama, Comedy, and International content.
- Most movies fall within a standard duration range (80–120 minutes).
- A few extreme outliers exist in movie durations (very short or very long content).
- TV Shows are fewer but play an important role in user retention and engagement.

---

## 📌 Conclusion
Netflix follows a balanced content strategy where **Movies are used for content volume and acquisition**, while **TV Shows are used for long-term user engagement and retention**. The platform also shows a strong global content strategy with diverse genres.
